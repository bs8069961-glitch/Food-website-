# Food-website
Food Website

A modern and responsive food website built using HTML, CSS, and JavaScript. This project features a clean user interface, smooth scrolling, interactive navigation, animated sections, and a responsive layout that works across desktops, tablets, and mobile devices.

 Features
Responsive design for all screen sizes
Fixed navigation bar
Mobile-friendly navigation menu
Animated hero section
Food speciality section
Popular food cards with ratings
Food gallery with hover effects
Customer reviews section
Order form
Scroll-to-top button
Loading screen animation
Smooth scrolling experience
Technologies Used
HTML5
CSS3
JavaScript (ES6)
Font Awesome Icons
Google Fonts (Nunito)
